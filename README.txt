# 🎭 Vibe Check

A streamlined AI tool that combines OCR capabilities with conversation analysis in an efficient, focused interface. Get quick insights into the emotional tone and dynamics of your communications.

## 🚀 Key Features

- One-click image-to-text conversion
- Purpose-built interface for conversation analysis
- Consistent, reliable feedback
- Clean, focused workflow
- Accessible web interface

## 🛠️ Technologies

- Python
- Streamlit
- OpenAI API
- OCR (Optical Character Recognition)
- Environment management

## 🎯 Use Cases

- Quick analysis of screenshot conversations
- Batch processing of multiple texts
- Emotional tone checking
- Team/shared access via web interface
- Professional communication review

## 🔧 Setup

1. Clone the repository
2. Create `.env` file with your OpenAI API key
3. Create `.streamlit/secrets.toml` with password
4. Install requirements: `pip install -r requirements.txt`
5. Run: `streamlit run main.py`


## 🔒 Security

- Secure API key management
- Password protected interface
- Environment variable protection
- Safe deployment practices
